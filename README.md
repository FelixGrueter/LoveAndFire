# Love & Fire

Landing Page fuer Love & Fire Spiritual Intensives — Events von MANTD in Basel, Waldshut und Villingen.

## Tech Stack

- HTML5, Tailwind CSS (via CDN), Vanilla JavaScript
- AOS (Animate On Scroll via CDN)
- Google Fonts: Plus Jakarta Sans, Manrope
- Material Symbols (Icons)

## Projektstruktur

```
index.html              Hauptseite
assets/images/          Bilder (Logo, Staedte, Event-Fotos)
assets/videos/          Video (Worship)
```

## Lokal starten

```bash
npx serve .
```
